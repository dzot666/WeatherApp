# WeatherApp

Тестовое задание: получение прогноза погоды с использованием [WeatherAPI.com](https://www.weatherapi.com/).

## ?? Описание задачи
Приложение получает прогноз погоды на **следующий день** для городов:
- Chisinau
- Madrid
- Kyiv
- Amsterdam

Данные выводятся в виде таблицы в консоль:

- минимальная температура (°C)  
- максимальная температура (°C)  
- влажность (%)  
- скорость ветра (км/ч)  
- направление ветра  

## ?? Используемые технологии
- Kotlin  
- OkHttp (HTTP-клиент)  
- org.json (парсинг JSON)  
- Gradle
